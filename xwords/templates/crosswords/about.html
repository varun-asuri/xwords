What is Xwords?

This website is an interactive website made by students of Dr. Gabor at TJHSST in Northern Virginia. In this website, you can solve American Crosswords of any size or shape with any number of blocks in 5 different languages.

This website runs on the Django Web Framework with a backend entirely composed of Python. Additionally it uses the Django Database to store word data and HTML to display pages. It incorporates Bootstrap CSS files for an appealing look.
